# 💰 Daily Budget - 오프라인 가계부 앱

Flutter를 사용하여 만든 **오프라인 가계부 앱**입니다. 이 프로젝트는 SQLite를 기반으로 동작하며, 인터넷 연결 없이도 안전하게 가구의 수입과 지출을 관리할 수 있습니다.

## 📱 프로젝트 소개

이 앱은 Flutter의 성능과 SQLite의 안정적인 로컬 저장 능력을 결합하여 빠르고 직관적인 가계부 경험을 제공합니다.

---

## 🚀 주요 기능

- ✅ **수입 / 지출 입력**: 간편한 UI를 통해 실시간으로 수입과 지출을 기록합니다.
- ✅ **SQLite 로컬 저장**: 모든 데이터는 기기에 오프라인으로 저장되어 개인정보 유출 걱정이 없습니다.
- ✅ **프리미엄 UI/UX**: 슬라이딩 헤더, 애니메이션 토글 및 세련된 카드 디자인을 제공합니다.
- ✅ **실시간 통계**: 현재 잔액, 총 수입 및 총 지출을 한눈에 파악할 수 있는 대시보드.
- ✅ **스와이프 삭제**: 불필요한 내역을 간단한 스와이프로 삭제할 수 있습니다.

---

## 🏗️ 실행 방법

1.  **패키지 설치**:
    ```bash
    flutter pub get
    ```

2.  **앱 실행**:
    ```bash
    flutter run
    ```

## 📦 패키지 정보

- `sqflite`: 로컬 데이터베이스 관리
- `path`: 파일 경로 설정
- `intl`: 날짜 및 통화 포맷팅

---

## 📦 APK 빌드

릴리스 버전의 APK를 빌드하려면 다음 명령어를 실행하세요.
```bash
flutter build apk --release
```

---

## 🤖 GitHub Actions (자동 빌드)

`main` 브랜치에 푸시할 경우 자동으로 APK가 빌드되며, 빌드된 파일은 GitHub Action Artifacts에서 다운로드할 수 있습니다. 설정 파일은 `.github/workflows/flutter_build.yml`에 위치합니다.

---

## 💡 핵심 정리

- **데이터 무결성**: SQLite를 통한 안전한 데이터 보관.
- **오프라인 우선**: 서버 응답을 기다릴 필요 없이 즉각적인 반응을 보장합니다.
- **현지화**: 한국어 로케일(`ko_KR`) 및 통화(`₩`)를 지원합니다.
